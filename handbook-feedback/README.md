# Handbook 反馈

本目录收集对 [AI x Web3 School Handbook](https://aiweb3.school/zh/handbook/) 的反馈、建议和勘误。

## 如何写反馈

创建文件，命名格式为 `YYYY-MM-DD-主题.md`，内容结构如下：

```markdown
# 反馈：[简短标题]

- 页面：[Handbook 页面链接]
- 类型：错别字 / 概念不清 / 信息过期 / 结构建议 / 内容缺失
- 日期：YYYY-MM-DD

## 问题描述
[具体哪里有问题或可以改进]

## 建议改法
[你的修改建议]

## 上下文
[你是在学什么、做什么时遇到这个问题的]
```

## 目的

- 帮助改进 Handbook，让后续学员更好地学习
- 记录自己的学习卡点和困惑
- 回馈开源学习社区
